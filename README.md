# Ingrid Aszkenasy Website

A contemporary art portfolio website inspired by minimalist design principles, featuring the work of Ingrid Aszkenasy.

## Features

- Clean, minimalist design with cursive handwritten header font
- Responsive layout that works on mobile, tablet, and desktop
- Six main pages: Home, Gallery, About, News, Shop, and Contact
- Artwork gallery with grid layout
- Contact form functionality
- Consistent navigation across all pages

## Structure

```
ingrid-v3/
├── index.html          # Home page
├── gallery.html        # Gallery page with artwork grid
├── about.html          # About page
├── news.html           # News page
├── shop.html           # Shop page
├── contact.html        # Contact page
├── css/
│   └── style.css      # Main stylesheet
├── js/
│   └── main.js        # JavaScript for navigation and interactivity
└── README.md          # This file
```

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- Vanilla JavaScript
- Google Fonts (Dancing Script for cursive header)

## Getting Started

1. Open `index.html` in a web browser
2. Navigate through the site using the navigation menu
3. All pages are linked and functional

## Customization

- Replace placeholder images with actual artwork images
- Update content in HTML files with your own text
- Modify colors and styling in `css/style.css`
- Add backend functionality to the contact form if needed

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
